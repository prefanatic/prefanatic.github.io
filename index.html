Projects (Past and Current)

Poro Snax

Poro Snax is a material design inspired League of Legends resource. Features include champion browsing, statistics, skins, and summoner information.

Hermes

Hermes is the backbone framework designed for the Wearable Internet of Things course at the University of Rhode Island. This framework helps accelerate the Android based research projects, and allows easier configuration of sensor data and data streaming.

Exsto

Exsto is a development framework for Garry's Mod. Exsto provided modular plugin through a load-unload approach, allowing developers to write and prototype extremely fast, without having to reload their game to see the changes. Exsto's primary focus was to provide administration features to server owners. Currently, Exsto is no longer maintained or functional.
